# uml-skill

UML-skill skill bundle for Cursor and other agents.

The main instructions live in **[SKILL.md](SKILL.md)** (`diagramming-uml`): copy this folder into `.cursor/skills/` or load `SKILL.md` as a project skill so the agent can author text-based UML and diagrams and, when **uml-mcp** is connected, call `generate_uml` / `validate_uml` for Kroki-backed URLs.
