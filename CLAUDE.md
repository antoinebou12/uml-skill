# uml-skill — Claude Code

This repo ships the **diagramming-uml** agent skill for text-based UML and diagrams (PlantUML, Mermaid, D2, BPMN, and more) with optional **[uml-mcp](https://github.com/antoinebou12/uml-mcp)** for Kroki URLs.

## What to read

1. **[SKILL.md](SKILL.md)** — complete instructions (YAML frontmatter + workflow). Start here for every diagram task.
2. **[README.md](README.md)** — how to install the skill (Smithery, Cursor, clone).
3. **[AGENTS.md](AGENTS.md)** — same read order as this file, plus deep links into the plan-then-generate sections of **SKILL.md**.

Optional on demand: **[references/DIAGRAM-TYPES.md](references/DIAGRAM-TYPES.md)**, **[scripts/README.md](scripts/README.md)** (Kroki URL helper without MCP).

## Smithery

Canonical listing: [diagramming-uml on Smithery](https://smithery.ai/skills/antoinebou12/uml). Install: `npx @smithery/cli@latest skill add antoinebou12/uml`.
